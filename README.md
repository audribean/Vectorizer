# Vectorizer
[# Demo Here](https://vectorizer-jisb.onrender.com/)

This is a web app that converts images into dxf files for the purpose of creating digital models out of drawings made by hand:
1. The user first uploads an image of a drawing on paper.
2. They then use the sliders to adjust what counts as marks on the page, all while previewing preliminary results.
3. Once satisified, the drawing can then be downloaded as a .dxf with a timestamp added to the filename.

This is a collaborative project of PabAudGal studios, with credit to Fergal Voorsanger-Brill for the initial idea and feedback.
